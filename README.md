# Nicotine Use Trends Interactive Chart

An interactive visualization of nicotine use trends from 1900 to 2024 across different product categories.

## Features

- Interactive stacked bar chart showing percentage trends for all nicotine types
- Hover functionality to display exact percentage values
- Click on legend items to toggle visibility of different nicotine types
- Combined bars for tobacco and non-tobacco products
- Responsive design with 4:3 aspect ratio

## Project Structure

- `index.html`: Main HTML file containing the chart implementation
- `cline_docs/`: Documentation directory
  - `projectRoadmap.md`: Project goals, features, and completion criteria
  - `currentTask.md`: Current objectives and implementation details
  - `techStack.md`: Technology choices and architecture decisions
  - `codebaseSummary.md`: Overview of project structure

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js

## How to Use

1. Open `index.html` in a web browser
2. Hover over data points to see exact percentage values
3. Click on legend items to toggle visibility of different nicotine types
